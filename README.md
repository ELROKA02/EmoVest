# 🧠 EMOVEST 🧠 — Emotional Invest

EMOVEST es una plataforma de análisis de trading orientada a eliminar el componente emocional de la inversión mediante análisis estadístico y procesamiento de lenguaje natural.

El sistema permite a los traders registrar sus operativas como un diario de trading tradicional y utiliza inteligencia artificial para detectar patrones emocionales y correlacionarlos con resultados financieros.

---

## 🎯 Objetivo del proyecto

La mayoría de los traders pierden dinero no por falta de estrategia, sino por errores emocionales.

EMOVEST busca:

- Reducir decisiones impulsivas
- Identificar patrones emocionales en el trading
- Convertir el comportamiento del inversor en datos analizables
- Mejorar la consistencia operativa mediante análisis estadístico

---

## 🧠 Cómo funciona

1. El usuario escribe sus operaciones en formato libre (diario de trading)
2. La IA analiza el texto para detectar:
   - Estado emocional
   - Sesgos cognitivos
   - Patrones conductuales
3. Se generan métricas estadísticas sobre:
   - Rendimiento vs emoción
   - Errores recurrentes
   - Evolución psicológica del trader
4. El sistema devuelve insights accionables
