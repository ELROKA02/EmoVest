# EmoVest 🧠

Plataforma web de trading enfocada en el análisis emocional y de datos para mejorar la toma de decisiones del inversor.

## Introducción

EmoVest es una aplicación digital diseñada para ayudar a los traders a mejorar su rendimiento mediante el registro estructurado de operaciones y emociones. La plataforma transforma factores psicológicos como el miedo o la impulsividad en métricas analizables, facilitando una operativa más disciplinada, racional y basada en datos.

El proyecto está orientado a crear una herramienta escalable que combine tecnología, análisis de comportamiento y visualización de información para optimizar la gestión emocional en entornos financieros.
